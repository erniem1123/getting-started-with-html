Lets say for example we need a table to demonstrate a point for reference on our website. We can use < table >
to achieve the necessaryactions to create a table within our code.

Month Day Cost Intrest
April 24th $12.97 7%



The code for this table looks like this:

< table align="center" cellspacing = "20" border="0" bgcolor="grey" >

< tr >
< th >Month< /th >
< th >Day< /th >
< th >Cost< /th >
< th >Intrest< /th >
< /tr >
< td >April< /td >
< td >24th< /td >
< td >$12.97< /td >
< td >7%< /td >
